### Requirements
- Nodejs
#### Install
Global
```
$ npm install -g typescript
$ tsc -v
```
##### Setup
Project
```
$ tsc --init
```
###### Basics
```
$ npm install --save-dev typescript
$ node_modules/.bin/tsc --init
$ node --save-dev
```
### Basic Commands
```
$ tsc -v
$ tsc <file name>
$ tsc -w
$ tsc <file name> -w
$ tsc --init
$ tsc *.ts -w
$ tsc --watch
```

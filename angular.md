### Requirements
- Nodejs
### Install
Angular CLI
```
$ npm install -g @angular/cli
$ ng version
```
### Setup
Project
```
$ ng new my-app
$ cd my-app
$ ng serve
$ ng serve --open
```
Components
```
$ ng generate component <name> or ng gc <name>
```
Simple component
```
$ ng generate component <name> --skip-tests --inline-style --inline-template
```
Services
```
$ ng generate service <name>
```
Guards
```
$ ng generate guard <name>
```

### Setup
```
$ npm install -g @angular/cli
$ ng --version
$ ng new my-app
$ cd my-app
$ ng serve --open
```
Components
```
$ ng generate component <name>
```
Services
```
$ ng generate service <name>
```
Guards
```
$ ng generate guard <name>
```

### Commands
Setup
```
$ mkdir src
$ cd src
$ npm init -y
$ npm i -S sergey
$ mkdir _imports
```
Tag
```
<sergey-import src="" />
<sergey-import src="">...</sergey-import>
<sergey-slot />
```

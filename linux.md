### Basic commands
#### Working with directories, files and file content
```
$ man man
$ man sudo
$ pwd
$ cd
$ ls
$ clear
$ mkdir
$ rmdir
$ file
$ touch
$ rm
$ cp
$ mv
$ head
$ tail
$ cat
$ echo
$ more
$ less
$ chmod
$ diff
$ telnet
$ ln
```
#### System information
```
$ uptime
$ free
$ ps
$ df
$ fdisk
$ lsblk
$ top
htop (package)
```
#### Networking
```
$ ifconfig
$ ip
$ netstat --help
net-tools (package)
```
#### Package manager
```
$ man apt
$ sudo apt update
$ sudo apt upgrade
$ sudo apt search <package name>
$ sudo apt install <package name>
$ sudo apt remove <package name>
```
#### Text editors
```
$ vi <file name>
$ vim <file name>
$ nano <file name>
$ emacs <file name>
```
#### systemctl commands
Handle apache2
```
$ sudo systemctl stop apache2
$ sudo systemctl restart apache2
$ sudo systemctl disable apache2
$ sudo systemctl enable apache2
```
